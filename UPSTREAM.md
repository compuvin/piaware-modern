## Upstream Base

This project is being reworked from a local SkyAware/PiAware web installation.

The live-view web application in this tree is based in part on FlightAware's
`dump1090` web interface, particularly the `public_html/` portion of the
upstream repository:

- https://github.com/flightaware/dump1090
- https://github.com/flightaware/dump1090/tree/master/public_html

This note is intentionally high-level for now. The exact list of inherited,
retained, modified, and removed upstream files will be documented later as the
tree is reduced and reorganized.

The repository currently includes GPL-2.0 license text in [LICENSE](LICENSE)
to reflect that upstream base while this cleanup and separation work is in
progress.
